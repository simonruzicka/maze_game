# 🧸 Interactive Kindergarten Learning App
*Interactive whiteboard game designed to develop logical thinking in preschoolers.*

[🇨🇿 Přejít na českou verzi](#-česká-verze)

### 📖 The Story Behind the Project
I developed this application while working at a kindergarten in Brno. The school had three main classes: **Veverky (Squirrels), Medvědi (Bears), and Ježci (Hedgehogs)**. 

To make learning engaging, I created a custom character for each class. The children play with their class mascots on interactive whiteboards. The primary goal of the game is to intuitively develop **early programming/algorithmic thinking** and improve the children's attention spans in a fun, playful environment.

### 🚀 Key Features
* **Custom Avatars:** Unique characters designed for each of the three classes.
* **Interactive Whiteboard Optimized:** Large UI elements and intuitive controls tailored for touch screens.
* **Educational Value:** Focuses on logic, step-by-step problem solving, and attention training.

# 🧸 Interaktivní výuková aplikace pro školky
*Hra pro interaktivní tabule, která rozvíjí logické myšlení u předškoláků.*

### 📖 Příběh projektu
Tuto aplikaci jsem vytvořil během své práce v mateřské škole v Brně. Školka měla tři třídy: **Veverky, Medvědi a Ježci**. 

Aby děti učení bavilo, vytvořil jsem pro každou třídu vlastní postavičku (avatara). Děti hrají hru se svým maskotem přímo na interaktivních tabulích. Hlavním cílem projektu bylo hravou formou rozvíjet **programátorské (algoritmické) myšlení** a trénovat jejich pozornost a soustředění.

### 🚀 Hlavní funkce
* **Třídní maskoti:** Unikátní postavičky na míru pro Veverky, Medvědy a Ježky.
* **Optimalizováno pro interaktivní tabule:** Velké UI prvky a ovládání přizpůsobené pro dotykové obrazovky.
* **Vzdělávací přesah:** Rozvoj logiky, řešení problémů krok za krokem a trénink pozornosti.
